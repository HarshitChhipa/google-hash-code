# Google Hash Code 2019
### Pizza Practice Problem - Javascript Solution

Our NodeJS solution for the practice problem, the results:

  - example => **12** / 15
  - small => **40** / 42
  - medium => **48226** / 50000
  - big => **885023** / 1000000

Run with NodeJS 9 `node index`
